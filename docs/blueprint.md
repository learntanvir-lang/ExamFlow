# **App Name**: ExamWise

## Core Features:

- Add Exam Modal: Modal to create new exams with details like name, date, and image URL.
- Exam Card Display: Display exams as cards with image, name, date, and custom items. Includes View and Edit modes.
- Customizable Items: Allow users to add various item types to exam cards: title+date, title+checkbox, countdown timer, eligibility status, payment status, and button with link.
- Exam Sorting: Sort exams by date, placing past exams at the bottom of the list.
- Data Storage: Store exam data and user-specific information in Firestore, supporting multiple users, exams, and items.
- User Authentication: Implement secure user authentication using Firebase Auth.

## Style Guidelines:

- Primary color: Steel Blue (#4682B4) for a calm and focused ambiance.
- Background color: Light Cyan (#E0FFFF), almost white, providing a clean, distraction-free surface for exam information.
- Accent color: Sky Blue (#87CEEB) to highlight important actions or dates.
- Headline font: 'Space Grotesk', sans-serif, for a modern, technical feel appropriate to academic content.
- Body font: 'Inter', sans-serif, for readability.
- Use minimalist icons to represent different item types and exam categories.
- Clean, card-based layout to clearly present exams.
- Subtle transition effects for mode switching (view/edit) and adding new items.
